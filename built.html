soffyn

A wellness app built specifically for Black women — and for anyone who's ever felt like the tools that were supposed to help them weren't made for the way they think or feel.

No streaks. No shame. No pressure. soffyn is a space for rest, reflection, and self-knowledge — built with neurodivergent-affirming design principles from the ground up.

// Hackathon Build

A11y.AI

An AI-powered alt text generator built at the Sitecore Symposium Hackathon. It uses Claude Vision to generate contextually accurate, screen-reader-optimized alt text from uploaded images.

Most AI alt text tools spit out generic descriptions. A11y.AI was built to understand context — not just what's in the image, but how it would be experienced by someone navigating by screen reader. In consideration for production adoption.
